## Quick Start

1. **Install Poe Search**:
   ```bash
   pip install poe-search
   ```

2. **Set up your token**:
   ```bash
   poe-search config set-token YOUR_POE_TOKEN
   ```

3. **Search your conversations**:
   ```bash
   poe-search search "machine learning"
   ```
